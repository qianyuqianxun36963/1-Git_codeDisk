## 命名文件夹内结构

我的github中，只保存 readme.md，并同级目录新建gitCode、srcCode文件夹。

**readme里保存github地址** 并介绍所有项目的笔记。

**srcCode 不做任何修改** 直接保存 git clone 下来的代码。

**gitCode 自己写的代码** srcCode里面的代码只是参考，练习在这里。
